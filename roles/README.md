# UCM Role Contracts

This directory contains starter role contract files for the UCM vNext runtime.

- `contracts/*.yaml` defines the machine-readable role contract.
- Each contract should stay aligned with [role-contracts.md](/home/eugene/git/ucm/role-contracts.md).
- Runtime loaders should treat these files as declarative inputs, not prompts.

These are v1 templates. The runtime does not load them yet.
