# Pulse Engine v0

Canvas 기반으로 만든 모바일 2D 게임 엔진 프로토타입입니다.

포함된 기능:

- 고정 timestep 게임 루프
- Scene / Entity 기반 런타임
- 터치 + 키보드 입력 라우팅
- 카메라 추적과 2D sprite-style renderer
- JSON prefab / level asset 로딩
- 즉시 플레이 가능한 데모 게임

실행:

```bash
cd mobile-engine
npm run dev
```

브라우저에서 `http://127.0.0.1:4312` 를 열면 됩니다.
